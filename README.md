# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

リモートリポジトリ：ネット上に配置して複数人で共有するためのリポジトリ
ローカルリポジトリ：開発者一人ひとりが使用するために自分のPC上に配置するためのリポジトリ

## プッシュとマージの違いは何でしょうか？

プッシュ：ブランチの変更内容をリモートのブランチに反映させる為のコマンド
マージ：別のブランチの作業内容(変更履歴)をブランチに取り込むこと

## コミットとプッシュの違い

コミット：変更内容を保存する為のコマンド
プッシュ：別のブランチの変更内容をリモートのブランチに反映させる為のコマンド

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

一目で変更内容内容がわかる

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

ローカルでマージするフロー

①【ローカル】作業用ブランチで作業してコミット
②【ローカル】メインブランチにマージ
③【ローカル】メインブランチから【リモート】メインブランチにプッシュ

プルリクエストでマージするフロー

①【ローカル】作業用ブランチで作業してコミット
②【ローカル】作業用ブランチから【リモート】作業用ブランチにプッシュ
③【リモート】プルリクエスト
④【リモート】承認&マージ

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

１：先にマージされた変更内容を取り込む
２：後にマージしようとしている変更内容を取り込む
３：どちらの変更内容も取り込む
